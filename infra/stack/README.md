# Infrastructure Stack

This folder contains deployment-oriented infrastructure assets for local and cloud environments.

## Included services
- Docker Compose runtime
- NGINX reverse proxy
- PostgreSQL
- Redis
- MinIO
- Elasticsearch
- RabbitMQ
- Prometheus
- Grafana
- Loki
- Tempo
- Jaeger

## Operational goals
- health checks for services
- centralized logging and tracing
- backup planning
- secret management
- scaling guidance
