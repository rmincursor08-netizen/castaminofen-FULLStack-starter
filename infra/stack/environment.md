# Environment Variables

## Core runtime variables
- NODE_ENV
- PORT
- DATABASE_URL
- REDIS_URL
- RABBITMQ_URL
- MINIO_ENDPOINT
- MINIO_ACCESS_KEY
- MINIO_SECRET_KEY
- ELASTICSEARCH_URL
- JWT_SECRET
- APP_URL
- CDN_URL

## Environment conventions
- use `.env` for local development
- use secret managers for production values
- never commit secrets to source control
