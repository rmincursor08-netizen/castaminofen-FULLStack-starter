# Metadata and Optimization

## Media metadata
- store technical metadata such as codec, bitrate, resolution, duration, and format
- store business metadata such as title, language, copyright, tags, and content owner
- maintain a normalized metadata model in the database

## Optimization
- compress images to modern formats such as WebP and AVIF
- create multiple sizes for responsive delivery
- apply adaptive bitrate streaming for video
- generate audio previews and waveform data for quick UI rendering
