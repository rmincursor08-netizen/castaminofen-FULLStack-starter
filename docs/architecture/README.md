# Architecture Overview

This directory captures the system architecture, module boundaries, infrastructure choices, and technical standards for the platform.

## What to find here
- architecture summary and module map
- ADRs for major decisions
- developer and contribution guidance
- deployment, database, and API documentation

## Reading order
1. Start with the architecture overview and module map.
2. Review the ADRs for historical decisions.
3. Use the developer and contribution guides when changing the codebase.
