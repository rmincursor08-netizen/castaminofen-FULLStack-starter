# k8s

این پوشه برای فایل‌های Kubernetes و استقرار در محیط‌های مقیاس‌پذیر در نظر گرفته شده است.

## هدف
- نگه‌داری manifestها، Deploymentها، Serviceها، Ingressها و ConfigMaps
- آماده‌سازی پلتفرم برای استقرار در محیط‌های ابری یا روی‌هم‌رفته

## ساختار پیشنهادی
- k8s/base
- k8s/overlays/dev
- k8s/overlays/prod

## نکته معماری
استقرار باید declarative و قابل بازپخش باشد و وابستگی‌های محیطی از کد جدا شوند.
