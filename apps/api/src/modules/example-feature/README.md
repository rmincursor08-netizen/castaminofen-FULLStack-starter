# Example Feature Module

This folder demonstrates the required structure for an isolated feature module in the NestJS backend.

## Contents
- Controllers
- Services
- Repositories
- Entities
- DTOs
- Validators
- Events
- Commands
- Queries
- Interfaces
- Mappers
- Policies
- Guards
- Interceptors
- Filters
- Tests
- Factories
- Constants
- Errors
- Specifications
- Value Objects
- Aggregates
- Domain Events

## Rule
This module should remain self-contained and communicate with other modules only through interfaces or events.
