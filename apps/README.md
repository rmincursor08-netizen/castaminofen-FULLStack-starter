# apps

این پوشه برای برنامه‌های کاربرمحور در نظر گرفته شده است.

## هدف
- نگه‌داری اپ وب، اپ موبایل و ابزارهای کاربری
- جداسازی تجربه کاربر از منطق سرویس‌های پشتیبان

## ساختار پیشنهادی
- apps/web: اپ وب Next.js
- apps/mobile: اپ موبایل React Native / Expo
- apps/admin: کنسول مدیریت

## نکته معماری
هر اپ باید به‌صورت مستقل قابل توسعه و تست باشد و از بسته‌های مشترک استفاده کند.
