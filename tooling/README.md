# tooling

این پوشه برای ابزارهای توسعه و پشتیبانی پروژه در نظر گرفته شده است.

## هدف
- نگه‌داری ابزارهایی مثل linting، formatting، generation، CI helpers و task runners
- حمایت از توسعه‌دهنده و اتوماسیون فرآیندها

## ساختار پیشنهادی
- tooling/tsup
- tooling/eslint
- tooling/format
- tooling/ci

## نکته معماری
ابزارها باید از منطق کسب‌وکار جدا باشند و فقط در پشتیبانی توسعه و استقرار نقش داشته باشند.
