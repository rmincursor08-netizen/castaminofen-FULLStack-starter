# configs

این پوشه برای فایل‌های پیکربندی مشترک و تنظیمات سراسری پروژه در نظر گرفته شده است.

## هدف
- نگه‌داری تنظیمات TypeScript، ESLint، Prettier، Jest و ابزارهای کلان
- یک‌پارچه‌سازی تنظیمات میان همه‌ی اپ‌ها و بسته‌ها

## ساختار پیشنهادی
- configs/typescript
- configs/eslint
- configs/prettier
- configs/jest

## نکته معماری
تنظیمات باید در یک محل مرکزی نگهداری شوند تا تغییرات به‌صورت کنترل‌شده انجام شود.
