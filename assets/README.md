# assets

این پوشه برای فایل‌های ثابت، تصاویر، فونت‌ها، آیکون‌ها و منابع گرافیکی عمومی پروژه در نظر گرفته شده است.

## هدف
- نگه‌داری دارایی‌های چندرسانه‌ای و منابع تصویری قابل استفاده در اپ‌ها و مستندات
- فراهم‌سازی منبع مشترک برای طراحی و اجرای UI

## ساختار پیشنهادی
- assets/images
- assets/icons
- assets/fonts
- assets/logos

## نکته معماری
دارایی‌های ثابت باید از منطق برنامه جدا شوند و به‌صورت مدیریت‌شده نگهداری شوند.
